# Search Intent Ladder™

The Search Intent Ladder™ is BankDeMark’s framework for mapping content to buyer awareness stages.

Most websites target only transactional searches.

That is a mistake.

Google rewards sites that satisfy intent across the entire journey.

The Search Intent Ladder™ structures content into four layers:

---

## Layer 1 — Awareness

The user discovers a problem.

Examples:
- “why is my Shopify store not ranking”
- “why are aquarium plants dying”

Goal:
educate and attract attention.

---

## Layer 2 — Evaluation

The user compares solutions.

Examples:
- “best SEO tools for ecommerce”
- “daphnia vs baby brine shrimp”

Goal:
build trust and topical authority.

---

## Layer 3 — Transaction

The user is ready to act.

Examples:
- “buy live scuds Canada”
- “best Shopify SEO app”

Goal:
convert traffic into revenue.

---

## Layer 4 — Expansion

The user deepens engagement.

Examples:
- advanced guides
- email funnels
- related tools
- ecosystems
- communities

Goal:
increase retention and lifetime value.

---

# Core Principle

Most websites only build transaction pages.

Authority sites build intent ecosystems.

The Search Intent Ladder™ ensures every stage reinforces the next.
